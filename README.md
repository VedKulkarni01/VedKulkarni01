### 🔬 Featured Projects

**[EGFR Bioactivity Prediction](https://github.com/VedKulkarni01/egfr-bioactivity-ml)** — Ensemble ML pipeline (Random Forest, XGBoost, Neural Net) achieving 94.5% ROC-AUC on 20K+ compounds. Deployed as a Streamlit app with batch prediction for non-technical users.

**[COVID-19 snRNA-seq Lung Atlas](https://github.com/VedKulkarni01/covid19-snrnaseq-reproduction)** — Integrated 81K nuclei across 27 lung samples using Scanpy & scvi-tools. Identified immune cell subpopulations and 150+ dysregulated genes linked to inflammatory pathways. Fully reproducible with Docker + SLURM.

**[β-Cell scRNA-seq in Type 2 Diabetes](https://github.com/VedKulkarni01/scRNAseq-Beta-Cell-T2D-Analysis)** — End-to-end Nextflow pipeline (QC → STAR → featureCounts → DESeq2) profiling β-cell transcriptomes and revealing stress response signatures in T2D.

**[GWAS Interactive Dashboard](https://github.com/VedKulkarni01/potato-gwas-project)** — R Shiny dashboard for exploring GWAS results on 282 varieties with Manhattan plots, QQ plots, and population structure analysis for interdisciplinary teams.
